开发环境搭建

工具准备


"工欲善其事，必先利其器！"，在搭建开发环境之前，我们需要准备一些必要的开发工具和组件:

  • 1. MyEclipse或者Eclipse ,我个人比较喜欢MyEclipse，毕竟商业化的东西，更稳定一些。附：MyEclipse10破解工具下载
  • 2. Eclipse Maven插件。如果您安装的是MyEclipse，那么此项忽略。插件地址:http://www.apache.org/dyn/closer.cgi/maven/binaries/apache-maven-3.0.4-bin.tar.gz
  • 3. Eclipse SVN插件。插件地址：http://www.eclipse.org/subversive/latest-releases.php
  • 4. MySQL数据库。下载地址：http://www.mysql.com/downloads/，我用的是5.0.96版本。
  • 5. Tomcat。下载地址：http://tomcat.apache.org/，我用的是apache-tomcat-6.0.28。

搭建Puma开发环境

1. 在工程栏目空白处点击鼠标右键,选择import,或者选择MyEclipse菜单的File->Import

2. 找到SVN，选择Project from SVN

3. 选择Create a new repository location

4. 输入提供的SVN地址：https://v2.svnspot.com/boyuan2000cn.pumaframework/puma/

5. 输入SVN用户名和密码

6. 下一步

7. 选择用新向导创建工程

8. 选择Maven工程类型

9. 选择webapp类型

10. 输入group id， artiface id 以及package信息

11. 在导入的工程上右键，选择Mave4Eclipse->Update Porject Configuration

12. 在导入的工程上右键，选择属性，在弹出的窗口中找到Deployment Assembly.删除/target/WebRoot栏目

13. Puma framework开发环境搭建完毕！

1. 在工程栏目空白处点击鼠标右键,选择import,或者选择MyEclipse菜单的File->Import

2. 工程的导入和上面步骤类似，所不同的是选择mojo类型。
